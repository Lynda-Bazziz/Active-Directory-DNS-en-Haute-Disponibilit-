# Active-Directory-DNS-en-Haute-Disponibilit-
Ce projet consiste à mettre en place une infrastructure Active Directory Domain Services (AD DS) et DNS en haute disponibilité sous Windows Server.

L’objectif est d’éliminer le point de défaillance unique (Single Point of Failure) en ajoutant :

-un serveur secondaire répliqué
-un contrôleur de domaine en lecture seule (RODC)
